Resultados Reales en Nicaragua

Ingeniería & Preventa

React 19 / Gemini AI

PBS Control Digital (Suite OCR)

Problema Central: Dependencia de transcripción humana para cotizaciones B2B, márgenes calculados empíricamente y latencia operativa.

Solución Arquitectónica: Auténtica Suite "Full Cycle" impulsada por Gemini Flash. Escanea PDFs de proveedores (SINSA/SINTER) con OCR profundo, realiza *Fuzzy Matching* en milisegundos contra el catálogo corporativo (~400 ítems) para garantizar precios exactos, y exporta en formato estructurado Excel inyectando diseño corporativo ARGB.

Despliegue Técnico (Protocolo Aquiles v3.4):

  • 🛡️ Túnel Proxy Seguro: API Keys de Gemini protegidas tras Edge Functions en Supabase.
  • 🔌 Offline-First: Resiliencia local mediante IndexedDB (Dexie) para contextos sin red.
  • 🔐 Gobernanza RBAC: Políticas RLS (Row Level Security) que impiden el borrado no autorizado.

Logística Corporativa

Next.js / Android Native

GPS-PBS Tracker Híbrido

Problema Central: Pérdida del trazado de flotas por restricciones de "Deep Sleep" de Android, telemetría falsa y reportes kilométricos imprecisos.

Solución Arquitectónica: Plataforma consolidada (Web Admin + APK Conductor). La App Android opera como entidad "inmortal" rompiendo el ahorro de energía mediante Foreground Services y persistencia SQLite nativa. Incluye motor de IA meteorológico automatizado usando OpenWeatherMap cruzado con Gemini.

Despliegue Técnico (Life360 Level):

  • 📍 Filtro Kalman Dual: Rechazo de outliers (falsos saltos GPS) apoyado en giroscopio.
  • 🗜️ Robot Compresor Polyline: Reduce el peso de la base de datos PostgreSQL un 95% post-48 horas.
  • 📡 Pusher Nativo: Alertas push bidireccionales vía Firebase Cloud Messaging (FCM V1).

Operaciones Taller

React / TypeScript

SISTEMA CORE: TALLER PBS

Problema Central: Desconexión entre el registro de incidencias, los diagnósticos técnicos, repuestos de bodega y la generación de la cotización final.

Solución Arquitectónica: Monitoreo 360 del ciclo de vida de los equipos. Aplicación Web Progresiva (PWA) de alto rendimiento centrada en la "Visión TYM", permitiendo que técnicos, bodega y ventas interactúen sobre el mismo hilo de datos con sincronización Serverless.

Despliegue Técnico (PWA Engine):

  • ⚡ Arquitectura BaaS: Backend Serverless de alta concurrencia vía Supabase.
  • 📱 Ergonomía Táctil: Diseño Touch-First con sistema robusto de enrutamiento React Router.
  • 🔔 State Sync: Notificaciones no bloqueantes (Sonner) guiando el flujo operativo en bodega.

Control Logístico B2B

Node / Express API

Gestión de Partes de Valor Crítico

Problema Central: Trazabilidad rota en las importaciones de partes, tiempos de entrega (ETA) calculados al azar y falta de registro auditable en las aprobaciones.

Solución Arquitectónica: Sistema Monolítico Modular regido por el ciclo de vida estricto del repuesto: Solicitado → Aprobado → Proceso → Disponible. El Dashboard inteligente orquesta automáticamente retrasos logísticos y mueve los ítems a Backlog para mantener la pantalla libre de ruido operativo.

Despliegue Técnico (Seguridad Fail-Secure):

  • 🧱 Tipado de Hierro: Validación Zod Schema bloqueando peticiones hostiles al API REST.
  • ⏳ Motor ETA Automático: Algoritmo de inyección de fechas (50 vs 30 días) configurado vía BD.
  • 🔐 Auditoría Inmutable: Autenticación JWT forzada (6 hrs logout) con tracking de cambios "Quién/Cuándo".
